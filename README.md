### ❌ Минусы и проблемы

#### 🔴 Критические проблемы (исправить немедленно)

1. **Отсутствует sitemap.xml**
   - `robots.txt:4` ссылается на `sitemap.xml`, но файла нет
   - **Решение:** создать sitemap.xml или убрать строку из robots.txt

2. **Опечатка в CSS: гигантский padding**
   - `styles.css:660` — `.section { padding: 12rem 0; }` (192px!)
   - Должно быть `padding: 3rem 0;` или `padding: 2rem 0;`
   - **Решение:** исправить на адекватное значение

3. **Неиспользуемые файлы**
   - `images/card-lymphatic.svg` — не используется в HTML
   - `images/about-me-photo.svg` — дублирует другие файлы
   - **Решение:** удалить неиспользуемые файлы

#### 🟡 Средний приоритет

4. **Дублирование медиа-запросов**
   - `styles.css:2202-2246` — множественные `@media` с одинаковыми правилами
   - **Решение:** объединить в один блок

5. **Неконсистентные z-index**
   - scroll-progress: 400, header: 120, mobile-nav: 125, overlay: 110, cookie: 200
   - **Решение:** использовать систему (100, 200, 300, 400, 500)

6. **Хардкод цветов вместо переменных**
   - `styles.css:400` — `color: #1a1510`
   - `styles.css:926` — `#161d1a`
   - **Решение:** вынести в CSS-переменные

7. **Отсутствие debounce для resize**
   - `script.js:234` — `window.addEventListener("resize", ...)` без debounce
   - **Решение:** добавить debounce/throttle

8. **Магические числа в JS**
   - `script.js:482` — `y * 0.16` (коэффициент параллакса)
   - `script.js:485` — `y > 32` (порог для `.is-scrolled`)
   - **Решение:** вынести в константы с комментариями

9. **Валидация URL для Dikidi**
   - `script.js:116` — URL берется из атрибута без валидации
   - **Решение:** добавить проверку формата URL

#### 🟢 Низкий приоритет (опционально)

11. **Отсутствие preload для критических ресурсов**
    - Можно добавить `<link rel="preload">` для styles.css и script.js

12. **Нет Service Worker**
    - Для offline-режима можно добавить SW

13. **Emoji-favicon**
    - `index.html:25` — data URI с emoji
    - **Решение:** создать реальный favicon.ico + apple-touch-icon.png

14. **Отсутствие системы сборки**
    - Нет минификации CSS/JS
    - **Решение:** добавить Vite/Parcel для production build

15. **Пустые alt-атрибуты**
    - Изображения услуг имеют `alt=""` (декоративные)
    - **Решение:** добавить описательный alt для лучшего SEO
